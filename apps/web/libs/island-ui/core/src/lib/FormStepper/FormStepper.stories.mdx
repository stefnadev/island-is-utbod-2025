# FormStepper

<DescriptionFigma name="FormStepper" />

### Default

- Use `isComplete` to mark a section as complete.
- Use `isActive` to mark a section as active.

<Canvas>
  <Story name="Default">
    <FormStepperV2
      sections={[
        <Section
          isComplete
          section="Section #1"
          theme={FormStepperThemes.PURPLE}
        />,
        <Section
          isActive
          section="Section #2"
          theme={FormStepperThemes.PURPLE}
        />,
        <Section section="Section #3" theme={FormStepperThemes.BLUE} />,
        <Section section="Section #4" />,
        <Section section="Section #5" theme={FormStepperThemes.PURPLE} />,
      ]}
    />
  </Story>
</Canvas>

### With subsections

- Use `subSections` prop to `<Section />` to add subsections to a section.
- The `subSections` prop is an array of `ReactNode`s. You can use any component you want, e.g a `<Text/>` or `<Link/>`.

<Canvas>
  <Story name="With sub sections">
    <FormStepperV2
      sections={[
        <Section isComplete section={'Section #1'} />,
        <Section
          section={'Section #2'}
          isActive
          subSections={[
            <Text key="sub1" fontWeight="semiBold">
              Subsection #1
            </Text>,
            <Text key="sub2">Subsection #2</Text>,
            <Text key="sub3">Subsection #3</Text>,
          ]}
        />,
        <Section section={'Section #3'} />,
        <Section section={'Section #4'} />,
        <Section section={'Section #5'} />,
      ]}
    />
  </Story>
</Canvas>
